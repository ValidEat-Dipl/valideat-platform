# Persönliche Changes von Erik

Hier plant und dokumentiert Erik größere eigene Arbeiten. Vorlagen liegen unter `templates/`, laufende Changes unter `active/` und abgeschlossene Nachweise unter `completed/`.

Ein neuer Change erhält eine ID und einen kurzen englischen Namen, zum Beispiel `VAL-023-add-meal-entry`. Die persönliche Methode dazu ist in [../METHOD.md](../METHOD.md) beschrieben.
